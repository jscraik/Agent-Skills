# API limits and licensing

## Rate limits
- 1000 requests per hour, per user (24-hour period).
- Contact support to discuss temporary limit increases.

## Licensing
- All content in the Oak API is licensed under Open Government Licence v3.0 (OGL).
- OGL is compatible with Creative Commons Attribution 4.0.
- Content includes quizzes, slide decks, worksheets, videos, audio, and transcripts.

Source: https://open-api.thenational.academy/docs/about-oaks-api/api-limits
