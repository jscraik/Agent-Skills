# API overview

- Oak Curriculum API provides access to curriculum content across subjects and key stages 1-4.
- API access is free and requires an API key.
- The API includes endpoints for lessons, units, sequences, and curriculum metadata.
- Example endpoints referenced in the overview:
  - `/lessons/{lesson}/summary`
  - `/lessons/{lesson}/assets`
  - `/sequences/{sequence}/units`
  - `/key-stages/{keyStage}/subject/{subject}/questions`
- Use the API playground for interactive exploration.

Source: https://open-api.thenational.academy/docs/about-oaks-api/api-overview
