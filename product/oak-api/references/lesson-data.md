# Lesson data endpoints

- `GET /lessons/{lesson}/summary`
- `GET /lessons/{lesson}/overview`
- `GET /lessons/{lesson}/equipment-and-resources`
- `GET /lessons/{lesson}/anchor-questions`
- `GET /lessons/{lesson}/content-guidance`
- `GET /lessons/{lesson}/transcript`
- `GET /lessons/{lesson}/intro-quiz`
- `GET /lessons/{lesson}/starter-quiz`
- `GET /lessons/{lesson}/exit-quiz`
- `GET /lessons/{lesson}/lesson-quiz`
- `GET /lessons/{lesson}/video`
- `GET /lessons/{lesson}/presentation`
- `GET /lessons/{lesson}/worksheet`
- `GET /lessons/{lesson}/worksheet-answers`
- `GET /lessons/{lesson}/additional-materials`

Notes:
- Lesson assets are served from the API (e.g., videos, worksheets, presentations).

Source: https://open-api.thenational.academy/docs/api-endpoints/lesson-data
