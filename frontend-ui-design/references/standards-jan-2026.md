## Notes on status (Jan 2026)
- APG is a continuously updated resource hosted at WAI (not versioned as a TR).
- Core AAM 1.2 is a W3C Candidate Recommendation Draft (Dec 16, 2025);
  latest Recommendation remains Core AAM 1.1.

## Versioning notes
- WCAG 2.2 is a W3C Recommendation (12 Dec 2024) and ISO/IEC 40500:2025.
- WAI-ARIA 1.2 is the current Recommendation on W3C TR.
- Design Tokens 2025.10 is the first stable DTCG community group release.
- Design Tokens 2025.10 is a W3C Community Group Report (not a W3C Standard).
