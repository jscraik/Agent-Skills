# Versioning

- Versions represent the date the API was updated with new content or other updates.
- API uses semantic versioning for backward compatibility.
- Use versioned base URLs to stay on a stable version or move to the latest.
- Changelog endpoints:
  - `/changelog`
  - `/changelog/latest`
- Current major version is v0.

Source: https://open-api.thenational.academy/docs/about-oaks-api/versioning
