# Typography

## Intent
Use consistent text styles and truncation rules to maintain readability.

## Minimal pattern
- Use design system text components or Tailwind classes from tokens.
- Truncate with a tooltip or expansion when content can overflow.

## Pitfalls
- Avoid hard-coded font sizes when tokens exist.
- Avoid relying on color alone for emphasis.

## Accessibility
- Maintain contrast and readable line height.
- Use semantic elements (h1-h6, p) where possible.
