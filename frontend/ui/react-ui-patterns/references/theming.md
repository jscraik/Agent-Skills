# Theming

## Intent
Ensure components read from tokens and respond to theme changes.

## Minimal pattern
- Use CSS variables for runtime theming.
- Use Tailwind theme tokens where configured.

## Pitfalls
- Avoid hard-coded colors and spacing values.
- Avoid duplicating theme logic in components.

## Accessibility
- Verify contrast in light/dark modes.
- Provide high-contrast variants where required.
