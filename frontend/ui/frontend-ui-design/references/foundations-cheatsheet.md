# Foundations cheat sheet (condensed)

Last verified: 2026-01-01

This file mirrors essential tokens used by ChatGPT Apps UI foundations.

## Spacing scale (px)
0, 2, 4, 8, 12, 16, 24, 32, 40, 48, 64, 128

Guideline: use the scale consistently.

## Typography (SF Pro)
### Web
- heading1: 36 / 600 / line 40 / tracking -0.1
- heading2: 24 / 600 / line 28 / tracking -0.25
- heading3: 18 / 600 / line 26 / tracking -0.45
- body: 16 / 400 or 600 / line 26 / tracking -0.4
- body-small: 14 / 400 or 600 / line 18 / tracking -0.3
- caption: 12 / 400 or 600 / line 16 / tracking -0.1

## Color (hex)
### Light mode
- Background: #FFFFFF, #E8E8E8, #F3F3F3
- Text: primary #0D0D0D, secondary #5D5D5D, tertiary #8F8F8F
- Accents: blue #0285FF, red #E02E2A, orange #E25507, green #008635

### Dark mode
- Background: #212121, #303030, #414141
- Text: primary #FFFFFF, secondary #CDCDCD, tertiary #AFAFAF
- Accents: blue #0285FF, red #FF8583, orange #FF9E6C, green #40C977

## Iconography
Prefer outlined, monochrome icons that match the system style.
