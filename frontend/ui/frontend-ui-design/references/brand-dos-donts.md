# Apps-in-ChatGPT Brand Dos and Don'ts

Last verified: 2026-01-01

## Do
- Use flat, bordered surfaces with restrained elevation.
- Keep controls pill-shaped with 44px minimum height.
- Use rounded corners everywhere; avoid sharp edges.
- Use tight typography with consistent line-heights.
- Keep layouts dense but clear; prioritize content.
- Use semantic tokens for all values.

## Don't
- Don't introduce glossy gradients or heavy shadows.
- Don't use thin 1px borders where 0.5px is specified.
- Don't use placeholder-only inputs.
- Don't vary fonts away from SF Pro / SF Mono without explicit approval.
- Don't mix multiple visual styles in the same surface.
- Don't use color alone to convey meaning.
