# Management: connect

Source: https://developer.1password.com/docs/cli/reference/management-commands/connect/

Key points
- Connect commands manage 1Password Connect server resources.
- Use with `OP_CONNECT_HOST` and `OP_CONNECT_TOKEN`.

Examples
```bash
export OP_CONNECT_HOST="https://connect.example.com"
export OP_CONNECT_TOKEN="<token>"
op vault list
```
