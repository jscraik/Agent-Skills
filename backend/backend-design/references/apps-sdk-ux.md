# Apps SDK: UX Principles (Condensed)

## Principles
- Conversational leverage, native fit, composability.
- Extract core jobs; avoid porting full apps.
- Design for conversational entry and in-chat completion.
- Keep UI minimal; prefer structured responses.

## Checklist
- Conversational value, beyond base ChatGPT.
- Atomic tools with explicit inputs/outputs.
- End-to-end in-chat completion for at least one task.
- Performance and discoverability.
