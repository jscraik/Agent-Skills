# Contrast Minimums (Brand Baseline)

Last verified: 2026-01-01

Use these as minimums across all surfaces.

## 1) Text
- Normal text: 4.5:1 minimum
- Large text (>= 18pt regular or 14pt bold): 3:1 minimum
- Disabled text: 3:1 where possible; never below 2.5:1

## 2) Icons and controls
- Icons conveying meaning: 3:1 minimum against background
- Focus indicators: 3:1 minimum against adjacent colors
- Borders used for affordance: 3:1 minimum

## 3) States to verify
- Default, hover, pressed, focus, disabled, selected
- Light and dark themes

## 4) Brand-specific checks
- White text on dark overlays: ensure >= 4.5:1 by adjusting overlay opacity.
- Accent blues: ensure >= 3:1 against bg.primary and bg.tertiary.
