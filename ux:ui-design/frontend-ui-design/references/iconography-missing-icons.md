# Apps-in-ChatGPT Iconography (Supplemental / Missing Icons)

Last verified: 2026-01-01

Use these only when the official icon set does not cover a needed glyph.
Before adding new glyphs, check the official catalog:
`iconography-apps-in-chatgpt.md`

## Rules
- Keep stroke and proportions consistent with the official set.
- Default size 24x24, 2px stroke, round caps and joins unless noted.
- Avoid duplicates of existing icons; prefer aliases if identical.
